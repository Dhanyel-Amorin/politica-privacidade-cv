Política de Privacidade

Última atualização: 24 de abril de 2026

1. Introdução

Esta Política de Privacidade descreve como o aplicativo de conversão de vídeos trata as informações dos usuários. O Aplicativo tem como única finalidade converter arquivos de vídeo do formato .avi para .mp4.

Ao utilizar o Aplicativo, você concorda com os termos desta Política.

2. Coleta de Informações

O Aplicativo não coleta dados pessoais identificáveis dos usuários.

  • Não solicitamos nome, e-mail ou telefone
  • Não criamos contas de usuário
  • Não coletamos dados sensíveis

Arquivos de vídeo

  • Utilizados exclusivamente para conversão
  • Não são armazenados permanentemente
  • Não são compartilhados com terceiros

3. Uso das Informações

Os arquivos fornecidos são utilizados apenas para converter vídeos de .avi para .mp4.

4. Armazenamento de Dados

  • Os arquivos podem ser processados temporariamente durante a conversão
  • Após a conversão, são automaticamente excluídos

5. Compartilhamento de Dados

  • Não compartilhamos dados com terceiros
  • Não vendemos informações
  • Não utilizamos ferramentas de rastreamento

6. Segurança

Adotamos medidas razoáveis para proteger os arquivos durante o processamento. No entanto, nenhum sistema é totalmente seguro.

7. Permissões

  • Acesso a arquivos de mídia (para selecionar o vídeo .avi)
  • Acesso ao armazenamento (para salvar o .mp4 convertido)

8. Direitos do Usuário

Como não coletamos dados pessoais, não há dados armazenados para consulta ou exclusão. O usuário mantém total controle sobre seus arquivos.

9. Alterações nesta Política

Esta Política pode ser atualizada periodicamente.
